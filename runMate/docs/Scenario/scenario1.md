# Сценарий поиска пробежки

```plantuml
@startuml
actor User as "Пользователь"
participant App as "Приложение"
participant GeolocationService as "Сервис геолокации"

User -> App : Открывает приложение и авторизуется
User -> App : Нажимает на кнопку "Поиск пробежек"
GeolocationService -> User : Запрашивает геолокацию
User -> GeolocationService : Возвращает геолокацию
GeolocationService -> App : Возвращает геолокацию

alt Есть доступные пробежки
    App -> User : Отображает карту и список пробежек поблизости
    
    alt Пользователь использует фильтры
        User -> App : Применяет фильтры (пол, возраст, время и т.д.)
        App -> App : Фильтрация пробежек
    else Пользователь не использует фильтры
        App -> User : Отображает все доступные пробежки
    end

else Нет созданных пробежек
    App -> User : Выводит сообщение: "Нет пробежек. Предложить создать пробежку"
end

@enduml
